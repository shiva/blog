## Storyteller Tactic
**Order & Chaos** — the Known world: everyone focused on model capability (benchmarks, context windows, reasoning). The Unknown/Dragon: memory is already the binding constraint. Hero must navigate between them.

## Key Point
The model capability race is real but it has lapped the memory infrastructure race — and memory is now what decides whether an agent works in production.
