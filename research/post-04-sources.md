## Storyteller Tactic
**Man in a Hole** — agent starts in comfort zone (working well on session one), trigger (session ends), crisis (everything forgotten), recovery (user repeats everything), better place (if memory existed). 
The story structure: the agent that cannot remember is the agent that cannot improve.

## Key Point
An agent without persistent memory cannot learn. Every session is day one. The cost of this is not a one-time fee — it compounds.
