Well I confess, I'm a tool fanatic . Before you get werid ideas let me tell you that what I mean is that I like to use small, efficient utilities, that help me get my work done quickly. I have been on a perpetual hunt for **THE** best utility for almost all tasks.

A lot of my colleagues and friends seem to share this fascination for "those quick little things". So here is a list of some that I use quite regularly.

*   [w.bloggar ](http://wbloggar.com/) - Blog tool for Blogger.com

<p>*   [Putty](http://www.chiark.greenend.org.uk/~sgtatham/putty/) - SSH, SFTP client

<p>*   [KeyNote](http://www.tranglos.com/free/keynote_download.html) - Content Management software

<p>*   [CrapCleaner ](http://www.ccleaner.com/)- cleans unwanted files created by windows

<p>*   [ExplorerXP  ](http://www.explorerxp.com/)- File management

<p>*   [FileZilla ](http://filezilla.sourceforge.net/)  - Ftp client

<p>*   [Html-Kit](http://www.chami.com/html-kit/)    - HTML editor

<p>*   [JAlbum ](http://jalbum.net/)     - Photo album creator

<p>*   [Picasa ](http://www.picasa.com/)- Photo Organiser
